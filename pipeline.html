💡 Google Sheet Optimizer — Recommended Improvements

Your current "Pipeline Tier 1-3" sheet works, but these changes would make syncing cleaner and your day-to-day workflow faster. None of these are required — the dashboard works with the existing layout. These are suggestions if you want to upgrade the sheet itself.

1. Add a "Stage" Column

Replace the ambiguous "TIER 3 - call to be arranged" + "CALL ARRANGED?" columns with a single Stage column. Use values: Lead, Contacted, Discovery, Proposal, Negotiating, Signed, Lost. The dashboard can sync directly to the kanban column from this one field.

2. Add a "Priority" Column

Add a Priority column (values: Hot, Warm, Cold). This is the signal that's most missing from the current sheet — there's no way to see at a glance which prospects need action today vs. which are slow-burn.

3. Rename "LOCATION" to "CITY"

"LOCATION" is ambiguous — it sometimes contains city names, sometimes addresses. Renaming to CITY and having a separate ADDRESS column makes the data unambiguous and much easier to map.

4. Add "Estimated Annual Revenue" Column

Tracking estimated deal value per prospect makes the Analytics tab much more useful — you can see total pipeline value, not just count. Use a simple number like 4000 (for $4K/year).

5. Add "Last Contacted" Date Column

A Last Contacted date column lets the dashboard automatically flag prospects you haven't touched in 14+ days. Right now there's no way to sort or filter by recency.

6. Add a "Dashboard ID" Column

When syncing, the dashboard currently matches prospects by gym name (fuzzy). If a gym name ever changes or is spelled differently between rows, matches can break. Adding a stable Dashboard ID column (e.g., prospect_abc123) that the dashboard writes back gives you bulletproof sync matching. Optional but useful at scale.

7. Filter to Your Rows by Default

In the sheet, use Google Sheets' built-in Filter Views to create a saved view that filters column J (Sales Manager) = your name. This way when you open the sheet, you only see your rows — no need to scroll past other reps' leads.
